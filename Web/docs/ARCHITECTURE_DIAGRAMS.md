## 3. Authentication Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                        Login Flow                                │
└─────────────────────────────────────────────────────────────────┘

User enters credentials
         │
         ▼
┌─────────────────┐
│  Login Form     │
│  (email/pass)   │
└────────┬────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  POST /tenant/Users/login           │
│  Body: { email, password }          │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Backend validates credentials      │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Returns:                           │
│  - accessToken (JWT)                │
│  - refreshToken                     │
│  - user object                      │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Store in sessionStorage:           │
│  - accessToken                      │
│  - refreshToken                     │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Update AuthContext with user       │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Redirect to /dashboard             │
└─────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│                    Token Refresh Flow                            │
└─────────────────────────────────────────────────────────────────┘

API Request
         │
         ▼
┌─────────────────────────────────────┐
│  Request Interceptor                │
│  Check token expiration             │
└────────┬────────────────────────────┘
         │
         ├─────────────┬──────────────┐
         │             │              │
    Token Valid   Token Expired   No Token
         │             │              │
         ▼             ▼              ▼
    Add Token    Refresh Token   Redirect
    to Header         │           to Login
         │            ▼
         │    ┌──────────────────┐
         │    │ POST /refresh    │
         │    │ {refreshToken}   │
         │    └────────┬─────────┘
         │             │
         │             ▼
         │    ┌──────────────────┐
         │    │ Get new token    │
         │    └────────┬─────────┘
         │             │
         │             ▼
         │    ┌──────────────────┐
         │    │ Update storage   │
         │    └────────┬─────────┘
         │             │
         └─────────────┴──────────┐
                                  │
                                  ▼
                         ┌──────────────────┐
                         │ Continue request │
                         │ with valid token │
                         └──────────────────┘
```

## 5. SWR Cache Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                      SWR Cache Mechanism                         │
└─────────────────────────────────────────────────────────────────┘

Component Renders
         │
         ▼
┌─────────────────────────────────────┐
│  useSWR(key, fetcher)               │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Check cache for key                │
└────────┬────────────────────────────┘
         │
         ├──────────────┬─────────────┐
         │              │             │
    Cache Hit      Cache Miss    Stale Data
         │              │             │
         ▼              ▼             ▼
┌──────────────┐  ┌──────────┐  ┌──────────────┐
│Return cached │  │Call      │  │Return stale  │
│data          │  │fetcher   │  │data + refetch│
└──────┬───────┘  └────┬─────┘  └──────┬───────┘
       │               │                │
       │               ▼                │
       │         ┌──────────┐           │
       │         │API Call  │           │
       │         └────┬─────┘           │
       │              │                 │
       │              ▼                 │
       │         ┌──────────┐           │
       │         │Update    │           │
       │         │cache     │           │
       │         └────┬─────┘           │
       │              │                 │
       └──────────────┴─────────────────┘
                      │
                      ▼
              ┌──────────────┐
              │Component     │
              │re-renders    │
              │with data     │
              └──────────────┘
```

## 6. Request/Response Interceptor Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                    Request Interceptor                           │
└─────────────────────────────────────────────────────────────────┘

API Call Initiated
         │
         ▼
┌─────────────────────────────────────┐
│  Request Interceptor Triggered      │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Get accessToken from storage       │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Check if token is expired          │
└────────┬────────────────────────────┘
         │
         ├──────────────┬─────────────┐
         │              │             │
    Not Expired     Expired      No Token
         │              │             │
         ▼              ▼             ▼
    Use Token    Refresh Token   Continue
         │              │          without
         │              ▼             │
         │      ┌──────────────┐     │
         │      │Get new token │     │
         │      └──────┬───────┘     │
         │             │             │
         └─────────────┴─────────────┘
                       │
                       ▼
         ┌─────────────────────────────┐
         │ Add Authorization header:   │
         │ Bearer {token}              │
         └────────┬────────────────────┘
                  │
                  ▼
         ┌─────────────────────────────┐
         │ Send request to API         │
         └─────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│                   Response Interceptor                           │
└─────────────────────────────────────────────────────────────────┘

API Response Received
         │
         ▼
┌─────────────────────────────────────┐
│  Response Interceptor Triggered     │
└────────┬────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│  Check response status              │
└────────┬────────────────────────────┘
         │
         ├──────────────┬──────────────┬──────────────┐
         │              │              │              │
      200 OK         401 Error     Other Error    Network
         │              │              │           Error
         ▼              ▼              ▼              │
    Return Data   Refresh Token   Show Error        │
         │              │              │              │
         │              ▼              │              │
         │      ┌──────────────┐       │              │
         │      │Retry request │       │              │
         │      │with new token│       │              │
         │      └──────┬───────┘       │              │
         │             │               │              │
         │             ├───────────────┘              │
         │             │                              │
         │             ▼                              │
         │      ┌──────────────┐                      │
         │      │Success?      │                      │
         │      └──────┬───────┘                      │
         │             │                              │
         │      ┌──────┴──────┐                       │
         │      │             │                       │
         │     Yes           No                       │
         │      │             │                       │
         └──────┘             ▼                       │
                      ┌──────────────┐                │
                      │Redirect to   │◀───────────────┘
                      │login page    │
                      └──────────────┘
```
